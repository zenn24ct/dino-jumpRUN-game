# 🦖 Dino風ジャンプランゲーム

## 🎮 概要
- ブラウザで動く **1ファイル完結のジャンプランゲーム**
- スペースキー / ↑キー / タップでジャンプ
- 障害物を避けてスコアを稼ぐシンプルなランナーゲーム
- 「Dino（恐竜）ゲーム」風の軽量版
- PC・スマホどちらでもプレイ可能

---

## ⚙️ 操作方法
- **スペースキー / ↑キー / Wキー / タップ**：ジャンプ  
- **Pキー**：一時停止／再開  
- **Restartボタン**：リスタート  
- **パネルボタン**（Jump / Pause）でも同様に操作可能  

---

## 💻 技術構成
- **HTML / CSS / JavaScript** だけで構成（外部ライブラリなし）
- CSS変数（`--ground-h` など）で高さや色を簡単に調整可能
- **localStorage** でベストスコアを自動保存
- **requestAnimationFrame** によるスムーズなループ制御
- **getBoundingClientRect()** による正確な衝突判定

---

## 🧱 ファイル構成
index.html ← この1ファイルで完結させてる


---

## 🪄 ゲームの仕組み（ざっくり）
- ランナーは一定位置で待機し、ジャンプで上下に移動  
- 障害物が一定間隔で右から流れてくる  
- 衝突判定で **Game Over**  
- 時間経過でスコアが自動加算される

## 🧭 改修メモ（内部仕様）
- `GROUND_H`（JS）と `--ground-h`（CSS）は **常に同じ値にすること**  
  - ズレるとランナーや障害物が宙に浮いたり埋まったりする  
- 衝突判定は `getBoundingClientRect()` を使用  
  - `transform` で位置を変える場合、DOM更新後に呼ぶ  
- ランナーの上下移動は `transform: translateY()` を使用（滑らかで高速）
